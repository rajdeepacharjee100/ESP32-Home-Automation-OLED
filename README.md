# ESP32 Smart Home Automation with OLED Display

## Project Overview

This project is a smart home automation system using ESP32, relay module, and OLED display. It allows users to control appliances using voice commands via SinricPro and Google Assistant.

## Features

* Voice control using Google Assistant
* Real-time ON/OFF status on OLED display
* WiFi-based IoT control
* Fast and stable system design
* Clean UI with OLED display

## Components Used

* ESP32
* Relay Module
* OLED Display (GME12896-03)
* Jumper wires

## Connections

* Relay → GPIO 26
* OLED SDA → GPIO 21
* OLED SCL → GPIO 22

## Working

* ESP32 connects to WiFi
* User gives voice command
* SinricPro triggers relay
* OLED displays status (ON/OFF)

## Future Improvements

* Multi-device control
* Mobile app UI
* Power monitoring system
